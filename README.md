# Sabores de la Noche – Gastronomic Night Festival

A landing page for a fictional nocturnal gastronomic festival held across multiple cities in Colombia.

## Description

This project is a fully responsive landing page that promotes the "Sabores de la Noche" festival. It includes event dates, thematic food zones, a gallery, and contact information, all wrapped in a dark, warm-toned visual identity.

## Technologies Used

- HTML5 (semantic tags)
- CSS3 (Flexbox, Grid, Custom Properties, Media Queries)
- Google Fonts (Playfair Display + Lato)

## Project Structure

```
festival-landing/
├── index.html
├── css/
│   └── styles.css
└── README.md
```

## How to Run Locally

1. Clone or download this repository.
2. Open `index.html` directly in your browser — no server needed.
3. For best results, use a modern browser (Chrome, Firefox, Edge).

## Features

- Sticky header with hamburger menu on mobile
- Full-screen hero section with CTA buttons
- HTML table with event dates and cities
- Semantic list of festival experiences
- CSS Grid zones section with hover cards
- CSS Grid gallery with variable-span items
- Fully responsive: mobile, tablet, and desktop
