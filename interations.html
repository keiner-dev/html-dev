Aprende haciendo clic

Cada tarjeta tiene un botón funcional, el código que lo hace posible y la explicación línea por línea.

// 01

Cambiar el color del texto

Al hacer clic en un botón, JavaScript modifica la propiedad style.color del elemento. Pruébalo:

Este texto cambia de color

// Seleccionamos el elemento por su id
function cambiarColor(color) {
  const elemento = document.getElementById('texto-demo');
  elemento.style.color = color;
}
→ function cambiarColor(color) — define una función que recibe el color como parámetro.
→ document.getElementById('texto-demo') — busca en el HTML el elemento que tenga ese id.
→ elemento.style.color = color — cambia el color del texto directamente desde JavaScript.
→ onclick="cambiarColor('rojo')" — en el HTML, esto llama a la función al hacer clic y le pasa el valor.

// 02

Cambiar el tipo de letra

Con style.fontFamily puedes cambiar la fuente de cualquier elemento. Las fuentes del sistema siempre están disponibles.

El tipo de letra puede cambiar

function cambiarFuente(fuente) {
  const elemento = document.getElementById('texto-fuente');
  elemento.style.fontFamily = fuente;
}
→ style.fontFamily — en JavaScript las propiedades CSS con guión se escriben en camelCase: font-family → fontFamily.
→ Puedes pasar cualquier fuente del sistema: serif, sans-serif, monospace, cursive, fantasy.

// 03

Mostrar y ocultar un mensaje

Usamos style.display para mostrar u ocultar un elemento sin borrarlo del HTML.

¡Hola! Soy un mensaje oculto 👋
function mostrarMensaje() {
  const caja = document.getElementById('caja-mensaje');
  caja.style.display = 'block';
}

function ocultarMensaje() {
  const caja = document.getElementById('caja-mensaje');
  caja.style.display = 'none';
}
→ display: 'none' — oculta el elemento. Sigue existiendo en el HTML, solo deja de verse.
→ display: 'block' — lo vuelve visible como un bloque (ocupa su propia línea).
→ En el CSS el elemento arranca con display: none, así empieza oculto cuando carga la página.

// 04

Cambiar el fondo de toda la página

Con document.body accedemos al elemento <body> directamente y cambiamos su color de fondo.

El fondo de toda la página cambiará:

function cambiarFondo(color) {
  document.body.style.backgroundColor = color;
}
→ document.body — es el <body> de tu HTML. No necesitas getElementById porque siempre hay uno solo.
→ style.backgroundColor — la propiedad CSS background-color escrita en camelCase.
→ La transición suave ya está definida en el CSS del body con transition: background-color 0.4s ease.

// 05 · Bonus

Alternar con un solo botón (toggle)

En lugar de dos botones, un solo botón puede alternar entre dos estados usando una variable o classList.toggle.

¿Oscuro o claro?

// Variable que guarda el estado actual
let modoOscuro = true;

function alternarModo() {
  const texto = document.getElementById('texto-toggle');

  if (modoOscuro) {
    texto.style.color = '#111';
    texto.style.backgroundColor = '#f5f0e8';
    texto.style.padding = '0.5rem 1rem';
    texto.style.borderRadius = '8px';
  } else {
    texto.style.color = '#f0f0f0';
    texto.style.backgroundColor = 'transparent';
  }

  // Invertimos el estado para la próxima vez
  modoOscuro = !modoOscuro;
}
→ let modoOscuro = true — una variable que recuerda en qué estado estamos actualmente.
→ if / else — si es oscuro aplica estilos claros, si no, aplica estilos oscuros.
→ modoOscuro = !modoOscuro — el operador ! invierte el valor. true → false, false → true.
